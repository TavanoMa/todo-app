# ToDo App

Aplicação de lista de tarefas desenvolvida para gerenciamento completo de tarefas diretamente no navegador, utilizando **LocalStorage** para persistência de dados.

## Funcionalidades
- Adicionar novas tarefas
- Editar tarefas existentes
- Excluir tarefas
- Marcar tarefas como concluídas
- Fixar (pin) tarefas importantes
- Pesquisar tarefas por texto
- Filtrar tarefas (todas, concluídas e pendentes)
- Persistência automática dos dados

## Armazenamento de Dados
Todas as tarefas são salvas localmente no navegador utilizando **LocalStorage**, garantindo que os dados não sejam perdidos ao recarregar ou fechar a página.

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage
